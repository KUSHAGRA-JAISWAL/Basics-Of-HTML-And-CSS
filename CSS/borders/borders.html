Concepts of Using Borders in CSS!

Without Using border on the box

Borders add a border around your element.

it take three properties to set a border.

  • border - width
  • border - style
  • border - color
  • .box{
    border-color:.....;
    border-style:.....;
    border-width:.....;
    }
