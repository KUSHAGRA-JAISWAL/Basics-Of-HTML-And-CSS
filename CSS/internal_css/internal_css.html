Concepts of Using Internal CSS!

1. Inline css is annoying when you want to repeat the same changes is multiple places.

2. In general, inline css is never used.

3. we can use a stylesheet, rather then the inline styles, to make more global changes.

4. we can use internal style sheet that is placed in the head of our pages.

<-head->

<-style->

CSS Will GO Here

<- /style->

<- /head->

Selector

1. If i want to change the whole body of my page i can selsect and change like this:-

body {

color : #323232;

}

1. Always followed by opeaning curly brace -{.

2. The property of our element(s) that we wish to change.

3. The value we are assining to that property.

4. Dont forget the semi-colon ; at the end.

Selecing an element

Selecting an element is very easy.

1. Write the element without the < -->

2. Do not include attributes.

That's it !
