Concepts of Using Class in CSS!

Class Selector

  • Classes are a type of attribute we can add to an HTML element
  • < p class = "intro" >........Learning........< /p>
  • In CSS, the class attribute is refrenced by using a full stop.
  • .intro{
    font-size : 18px;
    color : #2b5dad;
    }
  • you can include the element before the classname, but you dont have to ( nor it is recommended ).
  • p.intro{
    font-size: 18px;
    color : #2b5dad;
    }
  • One class can be used at many places.
