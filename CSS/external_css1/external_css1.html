This is 1st Test file of External CSS

The Design of this web page will Changed with the External CSS file.

  • So here we will first add some text in html like this and then we will apply the css externaly.
  • One file controls all the styles, not the page you are on.
  • If you link the same CSS file with different html files then if you delete something from the stylesheet or change something it will change for all pages.

How to link External CSS to our HTML

  1. Make a External CSS file and save it with (.css)
  2. In you html inside the <-head->tag you have to all your external css.
  3. By writting : [< link href="your css file location" rel="stylesheet">] you can link your css with html.
  4. Here href contains the location of css file and rel means the relationship between documents here it is stylesheet.
