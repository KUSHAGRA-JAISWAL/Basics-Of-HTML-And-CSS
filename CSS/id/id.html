Concepts of Using ID in CSS!

ID Selector

  • IDs are a type of attribute we can add to an HTML element
  • < p id = "intro" >........Learning........< /p>
  • In CSS, the ID attribute is refrenced by using a hash tag.
  • #intro{
    font-size : 18px;
    color : #2b5dad;
    }
  • One ID can only be used at one place.
